# CasaClima

> Hub web de organizacao pessoal e residencial.

## Sobre o projeto

Aplicacao web multifuncional que centraliza ferramentas do dia a dia em um unico ambiente: previsao do tempo, controle financeiro residencial, noticias por categoria, rÃ¡dio online, cofre local de senhas, checklist da casa, contatos, notas e painel de perfil com IMC, sugestoes de treino e frase do dia. Dados persistidos localmente no navegador.

## Funcionalidades principais

- Clima atual e previsao dos proximos dias
- Painel financeiro com categorias, despesas e exportacao PDF/Word
- Noticias por categoria e busca de radios por regiao
- Cofre local, checklist da casa e cadastro de contatos
- Painel pessoal com IMC, treino e leituras sugeridas
- Persistencia local no navegador

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Vite
- Netlify Functions
- Playwright

npm install; npm run dev (ou abra dist/ apos build).
