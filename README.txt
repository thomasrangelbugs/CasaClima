# CasaClima

CasaClima é uma aplicação web multifuncional voltada para organização pessoal e residencial. O projeto centraliza ferramentas úteis para o dia a dia, como previsão do tempo, controle financeiro, notícias, rádio online, cofre local, checklist da casa, contatos, notas e painel de perfil.

## Funcionalidades

- Consulta de clima atual e previsão dos próximos dias
- Painel financeiro residencial
- Controle de renda, categorias, despesas extras e contas com vencimento
- Exportação de informações em PDF, imagem, Word e TXT
- Painel de notícias por categorias
- Cofre local para salvar acessos no navegador
- Busca de rádios por país, estado, cidade e gênero
- Favoritos de rádio
- Busca de serviços e lugares próximos
- Checklist da casa
- Cadastro de contatos importantes
- Notas e lembretes domésticos
- Painel pessoal com IMC, leitura astral, sugestões de treino, corrida, leituras e frase do dia
- Dados salvos localmente no navegador

## Objetivo

O objetivo do CasaClima é oferecer uma plataforma simples e prática para auxiliar na organização da rotina, reunindo em um único ambiente recursos de clima, finanças, lazer, utilidades domésticas e acompanhamento pessoal.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Netlify
